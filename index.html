Chobitech.DemoKit

Chobitech.DemoKit is a lightweight, high-performance Unity package designed to help you create professional demonstration environments for your projects and assets with minimal effort.

Key Features

  • Dual Workflow Support: Seamlessly handle both Asynchronous (Task-based) and Coroutine-based demonstration logic.
  • Ready-to-Use UI: Includes a polished header, footer, description area, and a rich-text supported log system.
  • Smart Execution Management: Built-in support for cancellation tokens and lifecycle hooks (OnDemoCompleted, OnDemoCanceled).
  • Clean Architecture: Manage multiple demo cases within a single main scene without cluttering your project.

Simple Workflow

  1. Global Setup: Configure your project identity (Title, Icon, Color Tags) in GlobalDemoInfo.
  2. Inheritance:
    • Inherit from AsyncDemoBase for modern async/await logic.
    • Inherit from CoroutineDemoBase for traditional Unity IEnumerator workflows.
  3. Metadata: Attach IndividualDemoInfo to your demo component and define its specific name and description.
  4. Placement: Place your demo GameObjects under the Demo Container in the DemoMainScene.
  5. Run: Launch the scene, select your demo from the UI, and execute it.

Getting Started

Follow our Getting Started guide to build your first demo, or check the API Reference for advanced customization.

Dependencies

  • Required Packages:

    • Unity.TextMeshPro
    • Unity.2D.Sprite

  • For Package Development: To include Chobitech.DemoKit as a dependency in your own package, add the following to your package.json:

    "dependencies": {
    "com.chobitech.unity.demo-kit": "1.0.0"
}
