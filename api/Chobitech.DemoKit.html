Chobitech.DemoKit
Table of Contents

Namespace Chobitech.DemoKit

Classes

AsyncDemoBase

Base class for asynchronous demonstrations using Task and CancellationToken. Ideal for complex asynchronous logic, awaiting multiple operations, or integrating with Task-based APIs. This class inherits from DemoBase to provide metadata and logging functionality.

CoroutineDemoBase

Base class for demonstrations using Unity Coroutines (IEnumerator). Suitable for tasks that are tightly integrated with the Unity lifecycle or for users who prefer the standard StartCoroutine pattern. Inherits from DemoBase to provide metadata and logging functionality.

CustomColorTag

A ScriptableObject that defines a custom color mapping for text tags. Used to associate a specific string tag with a color for rich text processing within the DemoKit log system.

CustomColorTagExtensions

Provides extension methods for collections of CustomColorTag to facilitate color processing.

DemoBase

Base class for all demonstration components. Provides common functionality for metadata management, logging, and lifecycle hooks. This class is inherited by AsyncDemoBase and CoroutineDemoBase.

DemoKitExtensions

Provides a set of static extension methods used throughout the DemoKit system for color conversion, safe Unity Object execution, and functional programming patterns.

DemoKitLog

Provides a standardized logging interface for the DemoKit package. All logs are prefixed with a rich text tag and are conditionally compiled to be editor-only.

DemoKitUtils

Provides utility constants and shared properties for the DemoKit system.

DemoKitUtils.DefaultValues

Contains default strings and file names used during the demo generation process.

DemoKitUtils.PresetGUID

Contains constant GUID strings for identifying specific assets and folders within the project.

DemoOrchestrator

The central controller for the DemoKit system. It manages the lifecycle of demonstrations, coordinates between UI components (Header, Footer, Log, Description), and handles asynchronous execution and cancellation of demo processes.

GlobalDemoInfo

A ScriptableObject that stores global configuration and metadata for the entire DemoKit system. It centralizes information like titles, icons, and shared rich text color tags used by all demo scenes.

IndividualDemoInfo

A ScriptableObject that holds specific metadata for an individual demonstration. It stores the title and a detailed description to be displayed within the DemoKit UI system. Each DemoBase component requires a reference to this asset.
