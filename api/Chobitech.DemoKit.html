Chobitech.DemoKit
Table of Contents

Namespace Chobitech.DemoKit

Classes

AsyncDemoBase

Base class for asynchronous demonstrations using Task and CancellationToken. Ideal for complex asynchronous logic, awaiting multiple operations, or integrating with Task-based APIs.

AsyncDemoManager

Manages the execution lifecycle of an AsyncDemoBase. It handles task cancellation using CancellationTokenSource and ensures that lifecycle hooks are called in the correct order.

BaseDrawer

A base component that provides side-drawer functionality. It handles UI animations, positioning with margins, and state transitions (open/close).

CoroutineDemoBase

Base class for demonstrations using Unity Coroutines (IEnumerator). Suitable for tasks that are tightly integrated with the Unity lifecycle or for users who prefer the standard StartCoroutine pattern.

CoroutineDemoManager

Manages the execution lifecycle of a CoroutineDemoBase. It handles starting and stopping Unity Coroutines and ensures that lifecycle hooks like OnDemoCompleted() and OnDemoCanceled() are invoked.

CustomColorTag

A ScriptableObject that defines a custom color mapping for text tags. Used to associate a specific string tag with a color for rich text processing.

CustomColorTagExtensions

Provides extension methods for collections of CustomColorTag.

CustomColorTagHolder

Manages a collection of CustomColorTag objects and provides functionality to convert custom tags within text into Unity Rich Text format.

CustomTagController

Static controller responsible for parsing and converting custom text tags (e.g., [tag]content[/tag]). It uses regular expressions to identify tags and provides hooks for custom string conversion.

DemoBase

Base class for all demonstration components. Provides common functionality for metadata management, logging, and lifecycle hooks. This class is inherited by AsyncDemoBase and CoroutineDemoBase.

DemoDescriptionArea

Manages the description area of the UI, including global and individual demo information. It coordinates with a BaseDrawer component to handle expansion and icon state transitions.

DemoFooter

Represents the footer UI component of the DemoKit. Provides controls for selecting individual demos via a dropdown and executing them with a run button.

DemoHeader

Represents the header UI component of the DemoKit. Manages the display of the main demonstration title and its associated icon.

DemoKitExtensions

Provides a set of static extension methods used throughout the DemoKit system for color conversion and safe Unity Object execution.

DemoLogArea

Manages the log display area within the DemoKit. Handles log updates, auto-scroll behavior, and coordination with a BaseDrawer for UI transitions.

DemoManagerBase

Abstract base class for managing the execution lifecycle of a demonstration. It abstracts the differences between Task-based and Coroutine-based execution, providing a unified interface for the orchestrator.

DemoOrchestrator

The central controller for the DemoKit system. It manages the lifecycle of demonstrations, coordinates between UI components (Header, Footer, Log, Description), and handles asynchronous execution and cancellation of demo processes.

GlobalDemoInfo

A ScriptableObject that stores global configuration and metadata for the entire DemoKit system. It centralizes information like titles, icons, and shared rich text color tags.

IndividualDemoInfo

A ScriptableObject that holds general information about an individual demonstration. This is typically used to store metadata such as the demo's title and a detailed description for display in the UI.

RectTransformCachedMonoBehaviour

A base class for MonoBehaviours that frequently access their UnityEngine.RectTransform. It caches the RectTransform component during the Awake phase to improve performance by avoiding repeated GetComponent calls.

ScrollTextArea

A UI component that provides a scrollable text area with optional title support. It handles automatic/manual scrolling, text wrapping configurations, and detects user interaction through UserScrollDetector and UserScrollBarTouchDetector.

UserScrollBarTouchDetector

Detects touch or click interactions specifically for a scroll bar. Implements Unity's EventSystem interfaces to track the pressing state, which is useful for pausing auto-scroll behavior when the user interacts with the UI.

UserScrollDetector

Detects user-initiated scrolling or dragging within a UI area. This component triggers an event when the user interacts with a scroll view through dragging or mouse wheel input, allowing other systems to respond to manual navigation.

Structs

CustomTagController.CustomTagData

A data structure representing the components of a captured custom tag.
