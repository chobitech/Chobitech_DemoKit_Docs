### Overview

**Chobitech.DemoKit** is a lightweight, high-performance Unity package designed to help you build and distribute software demos. From initial environment setup to final implementation, **Chobitech.DemoKit** supports your development process with a focus on speed and ease of use.

---

### Installation

#### via Asset Store
- [**chobitech Asset Store page**](https://assetstore.unity.com/publishers/135416): Please add **Chobitech.DemoKit** to **My Assets** and import your project.

#### via Unity Package
1. Download the latest `.unitypackage` from [GitHub Release](https://github.com/chobitech/Chobitech_DemoKit/releases/latest).
2. Drag and drop the downloaded file into your Unity Project, or import it via the menu [Assets] > [Import Package] > [Custom Package] and select the `.unitypackage` file.


---

### Quick Start

After importing **Chobitech.DemoKit**, you can create your demo with the following steps:

1. **Open the Setup Window**: Go to [Window] > [Chobitech] > [DemoKit] > [Demo Setup Window], or [Project Settings] > [Chobitech] > [DemoKit Setup].
2. **Follow the Wizard**: Complete the prerequisites and configure your project.
3. **Implement your Demo**: Use the Core Library and utilities to build your demo features.
4. **Prepare Distribution**: Use the wizard's distribution tools to package your demo.
5. **Distribute**: Share your generated demo files with your audience.

For more detailed instructions and setup guides, please refer to the documentation:

- [Online Getting Started Guide](https://chobitech.github.io/DemoKit/getting-started.html)

### Requirements
